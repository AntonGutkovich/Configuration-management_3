# Configuration-management_3

## What is it?
Command-line tool for an educational configuration
language toml

## The syntax of the language to be converted

**Array**
```
[value, value, value, ]
```

**Names**
```
[a-zA-Z] + value
```

**Values**
- Numbers
- Arrays

**Declaring a constant**

```
def name = value
```

**Calculating the constant**

```
$name$
```
